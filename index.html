Preparation Steps

  1. Download the Python script:
    Download Script View Source
  2. Install Python on your system if you haven't already. You can download it from the official Python website.
  3. Install the necessary libraries. Open your terminal or command prompt and run: 
    pip install requests pyperclip tomli
  4. Execute the downloaded Python script and select your Minecraft mods folder when prompted.
  5. Follow the instructions provided by the script if needet.
  6. Copy the IDs collected by the script.
